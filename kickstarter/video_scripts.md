## General Production Notes

### Recording Tips
- **Resolution:** Record at 1920×1080 minimum. 4K if possible for Kickstarter's video player.
- **Browser:** Use a clean browser profile — no bookmarks bar, no extensions visible. Dark mode if available.
- **Mouse:** Move the cursor deliberately and slowly. Avoid erratic movement. Viewers need to follow.
